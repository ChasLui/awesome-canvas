## Canvas

### 定义

"HTML5 中添加的 HTML &lt;canvas> 元素可以通过 JavaScript 脚本绘制图形。例如，它可以用于绘制图表、制作照片合成、创建动画，甚至可以进行实时视频处理或渲染。" - [Mozilla 开发者网络](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API)

## 库

### 使用 canvas 绘图

- [Akihabara](http://www.kesiev.com/akihabara/) 是一个使用 JavaScript 和 canvas 标签制作像素游戏的 HTML5 游戏库。
- [Chart.js](https://github.com/chartjs/Chart.js) 是一个轻量级的 JavaScript 库，用于使用 HTML5 Canvas 元素创建动态和视觉上吸引人的图表。
- [ChemDoodle](http://web.chemdoodle.com/) 是一个开源化学和化学信息学工具包，其中使用 canvas 解决常见的化学相关任务，以各种不同方式显示分子。
- [d3](https://github.com/d3/d3) (或 D3.js) 是一个使用 Web 标准可视化数据的 JavaScript 库。D3 帮助你使用 SVG、Canvas 和 HTML 将数据变为现实。
- [EaselJS](https://createjs.com/easeljs) 是一个使 HTML5 Canvas 元素使用变得简单的 JavaScript 库。用于创建游戏、生成艺术和其他高度图形化的体验。EaselJS 是 CreateJS 的一部分 - 一组模块化库和工具，它们可以一起或独立工作，通过 HTML5 在开放 Web 技术上实现丰富的交互内容。
- [fabric.js](http://fabricjs.com) 在 canvas 元素之上提供交互式对象模型，还具有 SVG 到 canvas（和 canvas 到 SVG）解析器
- [iio.js](https://github.com/iioinc/iio.js) - 一个加速 HTML5 Canvas 应用程序创建和部署的 JavaScript 库
- [isomerjs](http://jdan.github.io/isomer/) - 一个用于 HTML5 canvas 的等距图形库
- [Javascript-Voronoi](https://github.com/gorhill/Javascript-Voronoi) - Fortune 算法的 JavaScript 实现，用于计算 Voronoi 单元
- [Konva](https://github.com/konvajs/konva) - Konva.js 是一个 HTML5 Canvas JavaScript 框架，通过为桌面和移动应用程序启用 canvas 交互性来扩展 2d 上下文。
- [Origami.js](https://github.com/raphamorim/origami.js) - 重新设计 canvas API 接口的 JS 库
- [p5.js](https://p5js.org) - p5.js 是一个用于创建图形和交互体验的 JS 客户端库
- [Paper.js](https://github.com/paperjs/paper.js) - Scriptographer 移植到 JavaScript 和浏览器，使用 HTML5 Canvas。
- [Pencil.js](https://pencil.js.org/) - 一个优秀的模块化 JavaScript 库，具有清晰的 OOP 语法和大量功能。
- [Pixi.js](http://www.pixijs.com) - 超快的 HTML 5 2D 渲染引擎，使用 webGL 并具有 canvas 后备
- [Processingjs](http://ejohn.org/blog/processingjs/) 是一个数据可视化编程语言。
    - [概述](http://ejohn.org/blog/overview-of-processing/)
    - [91 个基础演示](http://ejohn.org/apps/processing.js/examples/basic/)
- [Proton](https://github.com/a-jie/Proton) 是一个轻量级且强大的 JavaScript 粒子引擎。使用它可以轻松创建无数酷炫效果
- [Pts.js](https://ptsjs.org) - Pts 是一个用于可视化和创意编码的 JavaScript 库。
- [Rough.js](https://roughjs.com/) - Rough.js 是一个图形库，让你能够以手绘风格绘制
- [Scrawl-canvas](https://scrawl-v8.rikweb.org.uk/) - 轻松向网页添加多个响应式、可访问和交互式的 &lt;canvas> 元素
- [Sketch](https://github.com/soulwire/sketch.js) - 跨平台 JavaScript 创意编码框架
- [Three.js](http://threejs.org/) 是一个使 WebGL - 浏览器中的 3D 变得简单的 JavaScript 库，但是你可以[使用 canvas 而不是 WebGL 进行渲染](http://threejs.org/docs/#Reference/Renderers/CanvasRenderer)
    - [介绍](http://threejs.org/docs/#Manual/Introduction/Creating_a_scene)
    - [一些惊人的示例](http://threejs.org/examples/)
- [tsParticles](https://particles.matteobruni.it/) 是一个轻量级库，用于轻松创建粒子动画。它包括最常用 JavaScript 框架（ReactJS、VueJS、Angular 等）的即用型组件
- [Visualize](https://github.com/filamentgroup/jQuery-Visualize) 是一个 JQuery 插件，使用 HTML canvas 元素从表格数据创建图表和图形。
- [zDog](https://github.com/metafizzy/zdog) - 用于 canvas 和 SVG 的扁平、圆形、设计师友好的伪 3D 引擎
- [ZIM](https://zimjs.com/) - ZIM 是一个通用的 Canvas 框架，具有简单、强大的 JavaScript，让从初学者到专业人士的每个人都能编码创意。
- [zrender](https://github.com/ecomfe/zrender) - 一个轻量级 canvas 库，为 Apache ECharts（孵化中）提供 2D 绘制

### 用于其他目的，但仍使用 canvas
